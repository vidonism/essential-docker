
### Create GIT repo
1. git init
2. git commit -m "First commit"
3. git remote add origin https://github.com/vidonism/essential-docker.git
4. git push -u origin master


### Commit new changes
1. git info
2. git commit -m "First commit"
3. git remote add origin https://github.com/vidonism/essential-docker.git
4. git push -u origin master
